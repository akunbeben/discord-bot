# Discord Jobseeker BOT
A simple discord BOT that can search job information for you.

## Resources
- [karir.com](https://www.karir.com)
- [kalibrr.id](https://www.kalibrr.id)

## Available Commands
`!karircom` 

`!kalibr`

## Command parameters
- keyword: required

## Usage Example
`!karircom software engineer`

`!kalibr software engineer`
